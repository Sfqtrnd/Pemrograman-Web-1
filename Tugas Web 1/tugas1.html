HyperText Markup Languange

Dibawah ini adalah contoh penggunaan paragraf dalam html.
Berdasarkan wikipedia, bahwa html atau singkatan dari HyperText Markup Language adalah bahasa markah yang digunakan peramban untuk menafsirkan dan menulis teks, gambar dan bahan lainnya ke dalam halaman web secara visual maupun suara.
Dibawah ini merupakan link untuk dapat melihat penjelasan lebih mengenai html.

wikipedia

  • HTML merupakan sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser.
  • HTML berupa kode-kode tag yag menginstruksikan browser untuk menghasilkan tampilan sesuai dengan yang diinginkan.
  • HTML saat ini merupakan standar internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C).
Adapula subsript yaitu objek dibawah dan superscript yaitu objek diatas
Ciri-ciri dokumen html yaitu:
  • Ekstensi file berupa .htm atau .html,
  • Non case sensitive,
  • Terdiri atas tag-tag pembuka dan penutup (walaupun ada beberapa tag yang tidak memiliki penutup),
  • Tag-tag saling berpasangan & bersarang.
